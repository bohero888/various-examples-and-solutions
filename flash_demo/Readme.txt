    Flash Demo
    
  Aplikacja ta pokazuje w jaki sposob mozna w swojej aplikacji wykorzystac
  pliki Flasha.
  
  Do tego programu dolaczona jest kontrolka ActiveX Flash'a.
  Jezeli chcesz skompilowac program musisz w Delphi zarejestrowac kontrolke
  ActiveX. W tym celu z menu Components -> Import ActiveX. Nastepnie wcisnij
  przycisk Add i podaj lokalizacje kontrolki ActiveX Flasha. Nastepnie na
  liscie powinna pojawic sie nowa pozycja. Po zaznaczeniu jej nacisnij
  przycisk Install.
