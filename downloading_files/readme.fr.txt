DictInstall
L'installateur des dictionnaires d'OpenOffice.org
v0.9.x



INTRODUCTION
------------

Ceci est un petit assistant qui télécharge et installe les dictionnaires
pour le correcteur orthographique d'OpenOffice.org 1.0.

Il ne fonctionne actuellement qu'avec Microsoft Windows. Cependant puisqu'il a été écrit
en utilisant Borland Delphi, il devrait être possible de le porter vers Linux (en utilisant Kylix).
J'éspère. Encore à faire.


INSTALLATION
------------

Le programme n'a pas besoin d'être installé en permanence. Simplement dézipper le fichier .zip
dans un répertoire, puis double-cliquer sur 'dictinstall.exe'. Il ne reste plus qu'à suivre les
instructions sur l'écran pour télécharger et installer les dictionnaires.

Remarquez que les dictionnaires doivent préalablement être enregistrés depuis le site d'OpenOffice.org.
Pour les instructions à ce propos, voyez à :

    http://www.ooodocs.org/dictinstall/nexthelp.html


IMPORTANT
---------

Ce logiciel est en version beta ... il se pourrait qu'il maltraite votre installation d'OpenOffice.org
ou votre ordinateur. Autant que je sache, cela n'est encore arrivé à personne mais ... et bien vous
voici averti -- vous l'utilisez à vos propres risques.

RENSEIGNEMENTS TECHNIQUES
-------------------------

DictInstall est écrit en Borland Delphi 6. Le source est disponnible à partir de la page internet 
et est distribué sous la licence LGPL (veuillez s'il vous plait lire license.txt pour plus d'informations).
Il utilise les composants listés ci-dessous, dont vous aurez besoin pour télécharger et installer les sources
sous Delphi pour compliler DictInstall.

ICS
Internet Component Suite; une suite complète de composants TCP/IP.

    Disponible depuis: http://www.overbyte.be

TZip
Un composant zip non visuel écrit par Angus Johnson qui utilise les librairies zip d'Eric Engler,
qui sont basées à leur tour sur InfoZIP.

    TZip: http://rpi.net.au/~ajohnson/delphi/
    Eric Engler: http://www.geocities.com/siliconvalley/network/2114/

    InfoZIP: http://www.info-zip.org


Un merci aux créateurs de ces composants pour avoir aidé à DictInstall de voir le jour.
