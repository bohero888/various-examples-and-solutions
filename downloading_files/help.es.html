El Instalador de Diccionarios OpenOffice.org
Ayuda

Este es un archivo de ayuda provisional para el Instalador de Diccionarios OpenOffice.org. Actualmente, contiene información sobre cómo configurar DictInstall para modo local y con servidores proxy. Espero aumentarlo pronto para hacer un completo "manual" del programa.

Modo Local

DictInstallpuede ser ejecutado en "modo de instalación local". Esto permite a los diccionarios ser instalados en un disco duro en vez de Internet. Esto puede ser particularmente útil para mantenedores de CD-ROM que deseen proveer el Instalador de Diccionarios con todos los diccionarios como parte de sus CDs.

Configurando el modo local
Para configurar el modo local, es necesario descargar todos los diccionarios y colocarlos (como archivos zip) dentro de una carpeta local. Los diccionarios pueden ser descargados desde el sitio web de OpenOffice.org. Recomiendo colocarlos en una carpeta local dentro de la carpeta de DictInstall, aunque otros sitios también son posibles. (DictInstall actualmente soporta directorios relativos para esta carpeta, y total soporte para carpetas absolutas está siendo considerado.)

Necesitas también descargarte una lista de diccionarios para cada idioma que desees dar soporte (aquí están las listas disponibles actualmente Español Inglés, Alemán, Danés and Finlandés). Observa que puedes cargar sólo la que desees y usarla para todos los idiomas. Una vez que hayas descargado la lista de diccionarios, colócala en el mismo directorio que los diccionarios comprimidos.

Asumiendo que tienes una carpeta llamada local dentro de la carpeta de DictInstall, ahora necesitas hace unos pocos cambios en el archivo dictinstall.ini. Debajo de [config] hay dos líneas como estas: 

	#localpath=local\
	#locallist=dict.txt
Necesitas remover el símbolo almohadilla (#). Esto provocará que el programa busque en la carpeta localpath (en este caso local\) y encontrar la lista de diccionarios (en este caso dict.txt) disponible ahí.

Si eliges descargar la lista de diccionarios para múltiples idiomas, entonces será necesario puntualizar para cada una lista. Por ejemplo, si descargar para Inglés y Alemán los archivos dict.txt,necesitarás renombrarlos para cada uno de ellos y colocar sus nombres en el archivo dictinstall.ini. Si renombrar el archivo alemán a dict.de.txt, entonces, necesitarás añadir la línea locallist=dict.de.txt a la sección alemana ([de]) del archivo de configuración: 

	[de]
	listurl=http://www.ooodocs.org/dictinstall/de/dict.txt
	nexthelp=http://www.ooodocs.org/dictinstall/de/nexthelp.html
	locallist=dict.de.txt
De hecho, en la versión actual, la línea está disponible como un ejemplo, sólo necesitas eliminarla la almohadilla del comienzo de la línea también.

Cuando ejecutes el Instalador de Diccionarios, mirará dentro de la carpeta que has especificado y verá que hay una lista de diccionarios allí. Si es así, el programa deberá instalar los diccionarios directamente desde dicha carpeta. Por favor, observa que si el programa no es capaz de encontrar los diccionarios directamente de esa carpeta, automáticamente comenzará a descargar la lista de diccionarios a través de Internet.

Servidores Proxy

Desde la versión 0.9.3, DictInstall soporta el uso de servidores HTTP proxy. Esto debería permitir usar el programa en muchas redes de internet donde previamente no se podía. Observe en este momento, que el soporte proxy está muy poco testado: cualquier feedback positivo o negativo sobre esta característica será altamente apreciado..

Si estás usando un HTTP proxy para navegar por la web, y usas Internet Explorer, el programa funcionará transparente. Cuando se está ejecutando, busca tus configuraciones de Internet Explorer e intenta usarlas.

Si no usas Internet Explorer, o si tienes problemas con esto, necesitarás configurar el proxy manualmente. Para hacer esto, necesitas modificar una línea en el archivo de configuraciones dictinstall.ini. 

	proxy=auto
Cuando pones auto, el programa busca la configuración del proxy de Internet Explorer y la usa. Si has tenido problemas con esto y deseas desconectar el proxy proxy para configurarlo completamente, cambia esta línea a: 
	proxy=off
Si estás usando cualquier otro navegador o quieres configurar el proxy manualmente, entonces debes cambiar esta línea para que contenga el servidor y el puerto de tu servidor. Por ejemplo, si tu servidor proxy es en el ordenador myproxy y el puerto 8000, entonces debes cambiar la línea a: 
	proxy=myproxy:8000

Si tienes cualquier comentario o has tenido cualquier problema con la característica de servidor proxy, por favor envíame un correo-e.
