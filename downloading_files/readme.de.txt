DictInstall
Die OpenOffice.org Wörterbuch Installation
v0.9.x


EINLEITUNG
----------

DictInstall ist ein kleines Hilfsprogramm, welches die aktuell verfügbaren Wörterbücher
für OpenOffice.org 1.0 herunterlädt und diese auf Ihrem System installiert. 


Zur Zeit ist dieses Programm nur unter Microsoft Windows lauffähig. Da das Programm
jedoch mit Borland Delphi geschrieben wurde, sollte es möglich sein das Programm
auch (unter Verwendung von Kylix) auf Linux zu portieren. Ich bin zuversichtlich, dies
bald tun zu können. 


INSTALLATION
------------
Das Programm bedingt keine Vorinstallation auf Ihrem System. Entpacken Sie einfach
die .zip-Datei in einen Ordner und führen Sie das Programm mit einem Doppelklick auf
'dictinstall.exe' aus. Anschliessend können Sie einfach den Anweisungen des Programms
folgen, um die entsprechenden Wörterbücher herunterzuladen und zu installieren.

Beachten Sie, dass die Wörterbücher innerhalb von OpenOffice.org anschliessend noch
aktiviert werden müssen. Eine genaue Anleitung erhalten Sie unter:

    http://www.ooodocs.org/dictinstall/de/nexthelp.html

TECHNIK
-------

DictInstall wurde in Borland Delphi 6 geschrieben. Die Quellen sind auf der Homepage unter
den Lizenzbestimmung der LGPL (bitte lesen Sie auch die license.txt um weitere Informationen
zu erhalten) frei verfügbar. Bevor DictInstall kompiliert werden kann, müssen die
untenstehenden Komponenten innnerhalb von Delphi registriert werden.  

ICS
Internet Component Suite; ein vollständiges TCP/IP Paket.

    Erhältlich unter: http://www.overbyte.be

TZip
Ein non-visuelles Komprimierungs-Paket, geschrieben von Angus Johnson unter Verwendung
Eric Englers Zip-Bibliotheken - diese wiederum basierend auf InfoZIP.
  
    TZip: http://rpi.net.au/~ajohnson/delphi/
    Eric Engler: http://www.geocities.com/siliconvalley/network/2114/
    InfoZIP: http://www.info-zip.org

Danke an die Programmierer dieser Komponenten, durch welche die Entwicklung von
DictInstall erst möglich wurde!
