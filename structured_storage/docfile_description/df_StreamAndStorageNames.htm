DocFile Stream and Storage Names

There are a few limitations on what a stream or storage name may be. I'll discuss one way of dealing with this later on. For now here are the rules

  • No storage or stream may have a name longer that 31 chars long
  • No storage or stream name may contain the following characters
    !
    :
    /
    \
  • Don't use any char with an ordinal value below 32 as the first character. These chars are used for special purposes.
